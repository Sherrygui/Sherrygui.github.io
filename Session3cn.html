机器人工程学院

师生比例: 8:1
硬件 + 软件:
Arduino IDE, Sparkfun Redbot Kit, PiTop Laptop, and Python
适合年龄: 12–18 (以及从未学习过相关课程的学生）
课程日期:7.22–7.26 / 7.29–8.2
时间: 9:00AM–4:30PM
项目: Level 3 Camp（两周）

课程介绍

本课程将带领学生进入机器人和电子工程学科的精彩世界!学生们不仅能学习 到基本的编程和电子电路概念，还将了解到如何构建机器人。与此同时，学生们还将在RedBot Arduino平台上编程来控制机器人执行自动化任务，并探索每个传感器/执行器的功能。

在学生们学习了解了基本的编程和电路知识之后，从第二周开始，教师们将教授学生使用 树莓派和各种传感器来设计各自的独立项目。 在整个课程中，学生们将学习Python的基础知识，使用Pi进行物理计算的方法，以及如何通过Pi的互联网连接与其进行远程通信等 内容。 在课程结束时，学生可以将他们的课程材料带回家，我们鼓励学生在未来进行更深入的学习！

课程信息

学生将学会

  • 如何使用 Arduino 软件进行编程
  • 信号处理和电子电路的设计
  • 如何修正错误和解决逻辑问题
  • Raspberry Pi 的设置和使用
  • 命令行界面和 Python 编码方法
  • 物理计算

学生将完成

  • 一个可以自主执行任务的机器人!
  • 让机器人按照自己意愿行动的代码
  • 机器人的改进和测试
  • 综合运用 Python，Raspberry Pi 和网络的项目!
  • 基本的代码(可进行修改，能应用 到未来其他项目的代码)

课程内容

第一部分

介绍如何通过 Arduino 软件控制 LED 灯
介绍如何通过 Arduino 软件制作 LED 灯光秀
RedBot 机器人构建挑战
直行:探索如何让 RedBot 机器人移动
转弯:既然机器人可以向前和向后行驶，学生们就可以让机器人转弯

第二部分

使用 RedBot 蜂鸣器制作音乐
超声波传感器的障碍物检测
线条追踪 IR 传感器
伺服电机:什么是伺服电机?
介绍伺服电机是如何工作的

第三部分

了解什么是传感器融合并开始着手完成结课课题
机器人工程的未来发展方向
欢迎! Raspberry Pi 简介
电子工程概论:使用 Raspberry Pi 进行电子原型设计的基础知识
Python 编程简介

第四部分

将电子工程与编程结合起来
制作音乐!使用 Python 编码原始旋律
Twitter 输入/Pi 输出(标签显示和 Twitter 发布)

第五部分

连接 USB 摄像头的移动侦测器
定格动画制作器
DIY 制作亚马逊的 Alexa 产品
结课课题
用电路和 Python 制作钢琴
RetroPie，电子游戏模拟器
使用 Pi 进行语音识别
参加展示日
